# ДЕПХАБ Mini App

Мини-приложение для агрегации онлайн казино и ставок, построенное на Astro.

## Установка

```bash
npm install
```

## Разработка

```bash
npm run dev
```

Приложение будет доступно по адресу `http://localhost:4321`

## Сборка

```bash
npm run build
```

## Предпросмотр

```bash
npm run preview
```

## Структура проекта

- `src/pages/` - страницы приложения (index, tasks, settings)
- `src/components/` - переиспользуемые компоненты
- `src/styles/` - глобальные стили и переменные
- `src/data/` - данные для казино и заданий

## Особенности

- Темная тема
- Мобильная верстка (максимальная ширина 480px)
- Три основных экрана: Главная, Задания, Настройки
- Компонентная архитектура на Astro
- SCSS для стилизации
